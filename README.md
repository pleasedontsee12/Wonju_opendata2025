# 📊 청년 유입/정착 친화도 진단 및 정책 추천 플랫폼
> **제1회 원주시 공공데이터 활용 아이디어 공모전 출품작**</br>
> 데이터 기반으로 원주시의 청년 정착 여건을 진단하고, 생성형 AI를 활용해 맞춤형 정책을 제안하는 솔루션입니다.

## 📂 프로젝트 소개
* **팀명**: 원주율3.14
* **주제**: 청년 인구 감소 대응을 위한 데이터 기반 정착 친화도 진단 및 정책 제안
* **기획 의도**: 단순한 정량적 통계가 아닌, 청년들이 실제로 체감하는 생활 환경(주거, 문화, 의료 등)을 입체적으로 분석하여 **'청년 정착 친화도'**를 지수화하고, 이를 바탕으로 실효성 있는 정책을 수립하기 위함입니다.

## 🚩 1. 기획 배경
* **문제점**: 원주시의 청년 인구 감소 문제가 지속되고 있으나, 기존 정책은 지역별/분야별 특성을 정밀하게 반영하지 못하는 한계가 있었습니다.
* **필요성**: 청년들의 생활 환경을 객관적으로 진단할 수 있는 지표가 부재하며, 데이터에 근거한 합리적인 정책 우선순위 결정이 필요합니다.

## 💡 2. 핵심 솔루션
### 🔹 청년 정착 친화도 지수
* **정의**: 원주시 내 읍면동별 청년 생활 여건을 0~100점 척도로 수치화한 지표
* **구성**: 주거, 문화, 의료복지, 교육, 쇼핑, 이동, 치안 등 **7대 핵심 분야** 데이터를 융합하여 산출
* **기능**: 지역별 강점과 약점을 한눈에 파악하고, 상대적으로 소외된 지역을 시각화(Mapping)하여 보여줍니다.

### 🔹 생성형 AI 정책 추천
* 진단된 친화도 데이터를 바탕으로, 각 지역의 취약점을 보완할 수 있는 맞춤형 정책을 AI가 제안합니다.
* 예: "A동은 문화 시설 점수가 낮으니 청년 복합 문화 공간 조성을 최우선으로 추천"

## 🔄 3. 서비스 파이프라인
> **데이터 수집** ➡️ **분석 및 시각화** ➡️ **인사이트 도출**

1.  **Public Big Data**: 8개 범주(주거, 문화, 의료, 교육 등) 공공데이터 수집
2.  **Processing**: 청년 친화도 지수 모델링 및 수치화 계산
3.  **Visualization**: 읍면동별 격차 지도(Map) 시각화
4.  **AI Recommendation**: 생성형 AI를 활용한 지역 맞춤형 정책 도출

## 📊 4. 활용 데이터
> 총 8개 범주의 공공데이터를 융합하여 분석 모델을 구축했습니다.

| 분류 | 주요 활용 데이터 예시 |
|:---:|:---|
| **주거** | 청년 1인 가구 비율, 전월세 거래량 등 |
| **문화** | 문화시설 분포, 청년 예술인 활동 현황 |
| **의료/복지** | 인구 1만명당 병원 수, 심리상담 센터 접근성 |
| **교육** | 평생 교육 강좌 수, 도서관 이용률 |
| **편의** | 복합쇼핑몰 및 상권 데이터 |
| **치안/안전** | CCTV 설치 현황, 가로등 밀집도 |

## 🏅 5. 기대 효과 (Impact)
1.  **객관적 진단**: '정착 친화도' 수치화를 통해 청년 생활 환경을 정밀하게 진단합니다.
2.  **맞춤형 정책**: 뜬구름 잡는 정책이 아닌, 데이터가 가리키는 소외 지역과 취약 분야에 예산을 집중할 수 있습니다.
3.  **균형 발전**: 읍면동 단위의 격차를 해소하여 원주시 전체의 청년 삶의 질을 상향 평준화합니다.
